## How It Works

1. Page loads with animated flowers blooming (5.5 seconds)
2. Valentine prompt appears asking "Will you be my Valentine?"
3. Click "Yes" to reveal a personalized love letter
4. Click "Back" to return to the flowers
5. The "No" button playfully escapes on hover!

## Customization

To personalize the message:
- Edit the text in the `.message-content` section in `index.html`
- Change colors by modifying CSS variables in `style.css`
- Adjust flower colors by changing the `.flower__white-circle` background color

## Files

- `index.html` - Page structure and markup

- `main.js` - Interactive functionality
